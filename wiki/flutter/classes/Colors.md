## 举个例子

1. Color

    ```dart
    Color selection = Colors.green[400]; // 选择了中间的绿色.
    ```

2. ColorSwatch

    ```dart
    Container(
    color: Colors.blue, // 等同于 Colors.blue[500] 或 Colors.blue.shade500
    )
    ```
